# RachãoApp — Base criativa: cards Instagram (Presidente)

> Documento para produção de artes (Canva, Figma, Photoshop, IA).  
> **Público:** quem organiza pelada — o **Presidente**.  
> **Objetivo:** tráfego pago → cadastro trial → uso do wizard / primeira partida.  
> **Tom:** direto, brasileiro, “cansado de grupo bagunçado no WhatsApp”.

---

## 1. Brief da campanha

| Item | Diretriz |
|------|----------|
| **Persona** | Presidente da pelada (25–45 anos), organiza terça/quinta/sábado, cobra quadra, escala time no grupo |
| **Dor principal** | Confirmação de presença, vaquinha atrasada, escalação na pressa, placar no fim de semana |
| **Promessa** | Uma ferramenta só: convite → presença → escalação → jogo ao vivo → PIX |
| **CTA único** | **Cadastrar grátis** / **Teste 14 dias grátis** |
| **Não falar** | Dono de estádio, app nativo, cobrança automática de PIX |
| **Prova social (se usar)** | “Feito para pelada amadora no Brasil” — evite números inventados |

### Funil sugerido (carrossel 5–7 cards)

```
Gancho (dor) → Solução → Benefício 1 → Benefício 2 → Benefício 3 → Oferta trial → CTA final
```

### Formatos Instagram

| Formato | Tamanho | Uso |
|---------|---------|-----|
| **Feed quadrado** | 1080 × 1080 px | Carrossel, post único |
| **Feed retrato** | 1080 × 1350 px | Mais texto (4:5) |
| **Stories / Reels** | 1080 × 1920 px | Versão vertical de cada card |
| **Safe zone** | Margem ~80 px nas bordas | Texto e logo não cortados |

## 2. Kit visual (obrigatório em todas as artes)

### Cores

| Uso | Hex |
|-----|-----|
| Fundo | `#0f1b2d` |
| Card / painel | `#162236` ou `#1c2d45` |
| Destaque / CTA | `#e8530a` |
| Texto principal | `#e8edf3` |
| Texto secundário | `#7a8fa6` |
| Sucesso (check, confirmado) | `#22c55e` |

### Tipografia

| Elemento | Fonte | Peso |
|----------|-------|------|
| Headline do card | **Barlow Condensed** | Bold / ExtraBold |
| Corpo, bullets, CTA | **Inter** | Medium / SemiBold |
| Números (placar, preço) | **Inter** tabular | Bold |

### Elementos de marca

- **Logo** RachãoApp no canto (superior esquerdo ou inferior central no card final).
- **Tagline** (opcional, pequena): `Pelada noturna sob holofotes` — tracking largo, laranja ou cinza.
- **Gradiente de fundo** (sutil): mancha laranja `#e8530a` com 20–30% opacidade no canto superior direito + azul `#1c2d45` no inferior esquerdo (como na landing).
- **Ícones:** linha fina, estilo Lucide — laranja para destaque, branco/cinza para suporte.
- **Mockup de tela:** usar frame de celular escuro; screenshot real do app (dashboard, presenças, ao-vivo) quando possível.

### Estilo fotográfico (opcional)

- Campo society/grama à noite, holofotes, silhueta de jogadores — **sem** marcas de clubes profissionais.
- Overlay escuro 60–70% para texto legível.

## 3. Estrutura padrão de um card

Use este grid em **todos** os cards do carrossel:

```
┌─────────────────────────────────────┐
│  [Logo]              Card 2/7       │  ← opcional: indicador de slide
│                                     │
│  EYEBROW (laranja, caps, 12–14px)   │  ← ex: "PARA QUEM MANDA NA PELADA"
│                                     │
│  HEADLINE                           │  ← Barlow 48–72px, 2–4 linhas máx
│  (branco)                           │
│                                     │
│  [Ícone ou mockup central]          │
│                                     │
│  Bullet ou frase de apoio           │  ← Inter 18–22px, cinza
│  (1 linha preferencial)             │
│                                     │
│  ─────────────────────────────────  │
│  CTA pill: Cadastrar grátis →       │  ← só no último card OU discreto
└─────────────────────────────────────┘
```

**Regras de copy nos cards**

- Headline: **máx. 8 palavras** (ideal 5–6).
- Um benefício por card — não misturar vaquinha + escalação no mesmo slide.
- Segunda pessoa: **você**, **sua pelada**, **seu grupo**.

## 9. Checklist antes de publicar

- [ ] Logo legível em tamanho pequeno (preview do feed)
- [ ] Headline lida em 2 segundos
- [ ] Contraste texto/fundo (WCAG: branco em `#0f1b2d` OK)
- [ ] Não promete “PIX automático” nem “app na loja”
- [ ] Preço R$ 19,90 só no card de oferta (se mencionar)
- [ ] Link da bio = página de cadastro com UTM
- [ ] Versão 1080×1080 **e** 1080×1920 exportadas
- [ ] Carrossel: card 1 funciona sozinho no preview (gancho visível)
